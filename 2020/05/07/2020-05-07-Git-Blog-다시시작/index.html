Git Blog 다시 시작

정말 오랜만에 깃블로그를 작성한다…

처음 내가 공부한 내용을 남기려고 깃블로그를 만들었었다. 생각보다 글 하나 쓰는데 시간이 많이 소요되어서 다음에 쓰자… 쓰자 하다 보니 이렇게 시간이 지나버렸다.
특히 깃블로그를 운영한지 얼마 되지 않고 운이 좋게 인턴에 합격해서 시간가는줄 몰랐다…
그래서 오늘을 기점으로 다시 블로그에 공부한 내용을 포스팅하려고 한다.
현재 회사에서 C#을 하고 있기에 주로 포스팅은 C#, MVVM패턴이 대부분이고 이외에 OpenCV와 Python을 업로드하려고한다.

앞으로 공부하는 내용은 느리더라고 꾸준하게 블로그에 올리면서 블로그를 채워가야겠다.
